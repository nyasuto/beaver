⚙️ GitHub Actions のトラブルシューティング

🔴 問題: GitHub Pages デプロイが失敗する

症状

  • デプロイアクションが404エラーで失敗
  • feature ブランチからのデプロイ試行
  • branch protection rule によるブロック

原因

GitHub Actionsワークフローが main ブランチ以外からもデプロイを試行していたため、ブランチ保護ルールに違反していました。

解決方法

1. ワークフローファイルの修正
# .github/workflows/beaver.yml
deploy-pages:
  if: {{ `${{` }} needs.preflight.outputs.should_update == 'true' && github.ref == 'refs/heads/main' {{ `}}` }}
  needs: [preflight, build-and-test]
  # ... 他の設定
2. 条件の追加確認

以下の場所にも同様の条件を追加:

  • Line 148: Upload to GitHub Pages
  • Line 286: GitHub Pages deployment
  • Line 330: Final deployment step

予防策

  • ワークフローでのブランチ条件確認
  • プルリクエストでの事前テスト
  • デプロイ条件の明示的な定義

🔨 ビルドエラーのトラブルシューティング

🔴 問題: Go mod tidy エラー

一般的な原因と解決方法

1. 依存関係の競合
go clean -modcache
go mod tidy
2. プロキシ設定の問題
export GOPROXY=direct
go mod download
3. バージョン制約の確認
go mod graph | grep 問題のモジュール
go mod why 問題のモジュール

🟡 問題: テスト実行の失敗

チェックリスト

  • ✅ 環境変数の設定確認
  • ✅ テストファイルの命名規則
  • ✅ import パスの確認
  • ✅ モックファイルの生成

よくある解決方法

go test -v ./...
go test -cover ./...
go test -race ./...

🌐 API問題のトラブルシューティング

🔴 問題: GitHub API レート制限

症状

  • API calls exceeded error
  • 403 Forbidden responses
  • X-RateLimit-Remaining: 0

対策

1. トークンの確認
  • Personal Access Token の権限確認
  • Organization の制限確認
  • トークンの有効期限確認
2. リクエスト最適化
  • ページネーション効率化
  • 必要なフィールドのみ取得
  • キャッシュ機能の活用
3. レート制限監視
curl -H "Authorization: token YOUR_TOKEN" \
  https://api.github.com/rate_limit

⚙️ 設定問題のトラブルシューティング

🔴 問題: beaver.yml が読み込めない

チェックポイント

  • ファイルの存在確認: ls -la beaver.yml
  • YAML構文の確認: yaml-lint beaver.yml
  • 権限の確認: ls -la beaver.yml
  • 文字エンコーディング確認

修復手順

1. 設定ファイルの再生成
beaver init
# 既存の設定を beaver.yml.backup にバックアップ
2. 段階的な設定復元
  • 基本設定から開始
  • 一つずつ設定項目を追加
  • 各段階で動作確認

🟡 問題: 環境変数が認識されない

確認コマンド

echo $GITHUB_TOKEN
echo $OPENAI_API_KEY
env | grep -E "(GITHUB|OPENAI)"

設定方法

1. 一時的な設定
export GITHUB_TOKEN="your_token_here"
2. 永続的な設定
# ~/.bashrc または ~/.zshrc に追加
export GITHUB_TOKEN="your_token_here"
3. GitHub Actions での設定
# .github/workflows/beaver.yml
env:
  GITHUB_TOKEN: {{ `${{` }} secrets.GITHUB_TOKEN {{ `}}` }}
  OPENAI_API_KEY: {{ `${{` }} secrets.OPENAI_API_KEY {{ `}}` }}
